State Space Representation

状态空间方程的一般形式

其中，$x$ 是 $n$ 维状态矢量； $u$ 是 $p$ 维输入(或控制)矢量； $y$ 是 $q$ 维输出矢量；$A(t)$ 是 $n\times n$ 方阵，为系统内部状态的联系，称为状态矩阵或者系统矩阵；$B(t)$ 是 $n\times p $ 矩阵， 为输入对状态的作用，称为输入矩阵或控制矩阵；$C(t)$ 是 $q\times n$ 矩阵，为输出与状态之间的关系，称为输出矩阵； $D(t)$ 是 $q\times p$ 矩阵，表示系统的输入直接作用在输出的部分，称为直接传递矩阵。

当 $A(t), B(t), C(t), D(t)$ 均为常数时，上述状态空间方程退化为线性时不变方程
